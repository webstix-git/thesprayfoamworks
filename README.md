# The Sprayfoam Works — Next.js Site

Next.js 15 App Router migration of the Sprayfoam Works marketing site (originally exported as bundled HTML from Design Canvas).

## Routes

| Page | Route |
|------|-------|
| Homepage | `/` |
| Spray Foam Insulation | `/spray-foam-insulation` |
| About SFW | `/about` |
| Madison | `/madison` |
| Commercial | `/commercial` |
| Residential | `/residential` |

## Development

```bash
npm install
npm run extract   # Re-extract page HTML + images from source .html bundles
npm run dev
```

## Production build

```bash
npm run build
npm start
```

## Environment variables

Copy `.env.example` to `.env.local`:

```
NEXT_PUBLIC_SITE_URL=https://www.sprayfoamworks.com
```

## Vercel deployment

The project includes `vercel.json` with a build command that runs extraction before build. Deploy the repository root to Vercel with the Next.js framework preset.

## Project structure

```
src/
  app/              # App Router pages + SEO (sitemap, robots)
  components/       # StickyHeader (client), PageContent
  content/          # Converted page HTML (generated by npm run extract)
  lib/              # Site config and page helpers
  styles/site.css   # Original interaction styles (hover, animations)
public/images/      # Extracted images from source bundles
```

## Source bundles

The original bundled HTML files in the repo root are kept as the source of truth. Run `npm run extract` after updating them to regenerate `src/content/*.html` and `public/images/`.
